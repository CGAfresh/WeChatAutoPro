# 微信自动化祝福助手 Pro - 操作说明

本软件用于自动化发送微信节日祝福或特定日期消息，支持个性化称呼替换和随机文案功能。

---

## ⚠️ 重要注意事项 (必读)

1. **管理员运行**：为了确保软件能控制微信，请**右键点击软件 -> 以管理员身份运行**。
2. **微信状态**：
   * 运行前请确保 PC 端微信**已登录**。
   * 窗口**没有**最小化到右下角托盘（保持在任务栏可见即可）。
   * **关键步骤**：请将微信切换至**联系人界面**，之后再点击开始发送。
   * <img width="865" height="278" alt="image" src="https://github.com/user-attachments/assets/dd369c80-7987-4da8-916b-ec923e1bc6ae" />

3. **释放鼠标**：点击“开始自动发送”后，**请勿触碰鼠标和键盘**，直到弹窗提示“任务完成”。
4. **屏幕缩放**：如果发送异常，请检查电脑“显示设置”中的缩放比例，建议设置为 **100%** 或 **125%**。

## 🚀 快速上手指南

### 第一步：设置发送日期与文案

1. 打开软件，点击顶部 **【任务设置】** 选项卡。
2. 点击 **“+ 新建任务”**。
   * **输入日期**：格式必须为 `MM-DD`（例如 `01-05` 代表1月5日）。
   * **任务名称**：给自己看的备注，例如“张总生日”或“春节祝福”。
     <img width="1051" height="701" alt="PixPin_2026-01-05_16-35-57" src="https://github.com/user-attachments/assets/cb855699-a4b1-4b99-82ff-9f1d91635136" />

3. 在下拉框中选择刚才新建的任务。
4. 在下方文本框输入祝福语：
   * **一行一条**：发送时会随机从这里抽取一条。
   * **插入称呼**：使用 `{name}` 代表对方的尊称。
   * **示例**：
     > 祝{name}新年快乐，万事如意！
     > {name}你好，许久不见，祝您身体健康。
5. 点击 **“保存当前任务文案”**。

### 第二步：添加联系人

1. 点击顶部 **【联系人管理】** 选项卡。
2. 点击 **“+ 添加联系人”**。
   * **微信搜索名**：**必须精准**。这是你在微信搜索框里输入能唯一找到对方的名字（通常是微信备注）。
   * **自定义称呼**：你想在祝福语里怎么叫对方（例如“李总”、“王老师”）。
   * **逻辑说明**：如果文案是“祝{name}快乐”，自定义称呼是“李总”，发出去就是“祝李总快乐”。
3. 列表里支持删除不需要的联系人。

### 第三步：开始运行

1. 点击顶部 **【运行控制】** 选项卡。
2. 程序会自动检测电脑今天的日期。
   * 如果今天日期匹配到了你设置的任务（例如今天是 `01-05`，你刚好设置了 `01-05` 的任务），程序会提示 **“📅 命中任务”**。
   * 如果没命中，程序会提示不在任务列表中。
3. 点击 **“开始自动发送”**。
4. 程序会自动打开微信 -> 搜索联系人 -> 发送个性化消息。

## ❓ 常见问题排查

**Q1: 软件提示“无法连接微信”？**
* **A:** 请先手动打开微信 PC 端并登录。如果已登录，请尝试以**管理员身份运行**本软件。

**Q2: 为什么发出去的消息只有名字，或者发了两遍？**
* **A:** 这是因为电脑卡顿导致剪贴板延迟。请关闭过多的后台软件，或者重启电脑后再试。软件内部已针对此情况做了优化。
* **注意**：请确定微信是否切换到**联系人界面**。
  <img width="865" height="278" alt="image" src="https://github.com/user-attachments/assets/abc06857-73b7-452d-b704-665d103c4031" />


**Q3: 搜索不到联系人？**
* **A:** 请检查【联系人管理】里的“微信搜索名”是否由于多字、少字导致无法在微信里精确搜索到该用户。

**Q4: 我想手动修改配置文件？**
* **A:** 软件首次运行后，会在同级目录下生成一个 `config.json` 文件。你可以用记事本打开它，批量编辑联系人或文案（注意保持 JSON 格式正确），保存后重启软件即可生效。
